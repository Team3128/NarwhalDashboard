Directions

1. Wheel Circumference

   a. Mark the top of the front wheel with tape.

   b. Push the robot in a straight line for 100 inches and record the number of rotations.

   c. The Wheel Circumference is 100 divided by the number of rotations.

   d. Go to the main robot class and adjust the value titled wheel_circumference.

2. Speed Scalar

   a. Open up the Smart Dashboard using the search bar, click File, then New. Ensure the Right and Left Max Speed is displayed.

   b. Turn the throttle to maximum and run the robot forward at max speed. Record these values (these values can also be used for step 3).

   c. Divide smaller value by the greater value. Set the speedScalar in the main robot class.

   d. If the right value is greater, change setleft_scalar to setRightSpeedScaler, and vise versa.

3. Feet Forward

   a. If not already done so in step 2, open Smart Dashboard using the search bar, click File, then New. Ensure that the Right and Left Max Speed is displayed.

   b. If not already done so in Step 2, turn throttle to max and drive the robot around at full speed.

   c. Record the maximum speeds and divide 1023 by the lower of the two speeds.

   d. Change the value titled PID_kF in the main robot class.

4. Wheel Base

   a. Ensure the robot has enough space to go in a circle with a 5 foot diameter.

   b. Run WheelBase auto file by going to the Dashboard, selecting it from the Auto dropdown, and going to the Drivers Station and selecting Autonomous, then Enable.

   c. Wait for the robot to finish, click Disable, and check the log in the Drivers Station for the calculated Wheel Base.

   d. Select ArcTurn on Narwhal Dashboard and run it.

   e. Verify that the robot travels exactly 90 degrees.

   f. Adjust Wheel Base as necessary (greater Wheel Base equates to a greater degree turned).

   g. Go to the main robot class and adjust the value titled wheelBase.

5. PID Value

   a. Measure 100 inches and mark on the ground.

   b. Run the 100inchRun auto by going to the Dashboard, then selecting 100inchRun from the auto dropdown.

   c. Adjust the PID_kP value in the main robot class until the robot travels exactly 100 inches. Be sure to redeploy each time. A higher p value equals more distance.
